# Mind-Game
This is a simulation of the Mind game with cards. The player must pair all the numbers, then the game ends.
First of all, the player chooses the number of cards they want to play. For example, if the player chooses 8 cards, there are 4 pairs of numbers.
After that, the player chooses a row and a column to reveal a number. The next combo of a row and a column must reveal the same exact number. If the same number is revealed, the two numbers stay visible until the game ends. Else, the numbers stay hidden.
The player cannot choose a combo of a row and a column that has aldready picked.
Also, the game keeps track of the number of tries the player had to take to pair all the numbers. After a round the game asks the player if they want to continue. If so, they type yes, else they type anything than yes. 
When the game is finished, it prints the minimun number of tries (as the highscore) needed to finish the round.
